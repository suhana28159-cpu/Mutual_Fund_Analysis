# Mutual Fund Analytics

## Project Overview

This project was completed as part of my internship at Bluestock.

The objective of this project was to collect, clean, analyze, and visualize mutual fund data using Python, SQLite, and Power BI. The project includes ETL processing, exploratory data analysis, performance analytics, a recommendation system, and an interactive dashboard.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* SQLite
* Power BI
* Git & GitHub

## Project Features

* Data Cleaning and Processing
* SQLite Database Integration
* Exploratory Data Analysis (EDA)
* Sharpe Ratio Analysis
* Alpha & Beta Analysis
* Rolling Sharpe Analysis
* Mutual Fund Recommendation System
* Power BI Dashboard

## How to Run

Install dependencies:

```bash
pip install pandas numpy matplotlib
```

Run the pipeline:

```bash
python run_pipeline.py
```

Run the recommendation system:

```bash
python recommender.py
```

## Dashboard

Open the `.pbix` file using Power BI Desktop to view the dashboard.
